# Punctuation Option Walkthrough

## 実施内容

音声認識結果に句読点（、。）を自動的に付与するかどうかを選択できるオプションを実装しました。

## 変更点

### 1. 設定管理 (`config_manager.py`)

- デフォルト設定に `"add_punctuation": True` を追加しました。

### 2. GUI (`gui.py`)

- 「Add Punctuation (、。)」チェックボックスを追加しました。
- 設定の保存・読み込み処理を更新しました。

### 3. 音声認識エンジン (`stt_worker.py`, `stt_worker_persistent.py`)

- `model.transcribe` 実行時に、オプションが有効な場合は以下のパラメータを渡すように変更しました：
  - `initial_prompt="こんにちは、元気ですか。"`: モデルに句読点の使用を促します。
  - `condition_on_previous_text=True`: 前後の文脈を考慮して正確な句読点配置を行います。
